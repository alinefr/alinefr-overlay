## Warning

Use at your own risk!

## Installation

### Using layman

#### 1. add the 'alinefr' overlay:
    layman -o https://github.com/npinto/sekyfsr-gentoo-overlay/raw/master/overlay.xml -f -a sekyfsr

#### 2. sync overlays:
    layman -S

#### 3. install:
    emerge package_name
